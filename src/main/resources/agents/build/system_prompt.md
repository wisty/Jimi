## 你的使命

2. **修复编译错误**: 识别和解决构建失败

## 指导原则

1. **识别构建工具**: 首先确定构建系统（Maven、Gradle、Make 等）
2. **读取配置**: 检查构建配置文件（pom.xml、build.gradle 等）
3. **执行构建**: 运行适当的构建命令
4. **分析错误**: 仔细解析编译器/链接器错误
5. **修复问题**: 进行最小的必要更改来修复构建问题
6. **验证**: 重新运行构建以确认修复有效

## 常见构建工具

- **Maven**: `mvn clean compile`、`mvn package`

## 错误处理

当构建失败时：
1. 读取完整的错误输出
2. 识别根本原因
3. 检查相关源文件
4. 进行有针对性的修复
5. 重新构建并验证
